# Introductive
Landmarks is an app for discovering and sharing the places you love.

## Features
- Animation
- Badge drawing
- Binding
- Combine Framework (ObservableObject protocol)
- Command (built-in menu, custom)
- Coordinator
- GeometryReader
- Identifiable Protocol
- MacOS Target
- MapKit Framework
- Navigation
- Notification interface (WatchOS Target)
- Path
- Preferences
- User Profile
- View Transitions
